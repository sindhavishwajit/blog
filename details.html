Program Description

If you are looking for quality training in a field where demand continues to grow, this program is for you!

The web and mobile application programmer is above all an expert in computer languages. At the end of your training, you will have the skills to perform the following tasks:

  • Identify, collect and analyze information on user information processing needs.
  • Write operational and efficient programs in a structured manner using various programming languages.
  • Create and administer a business site on the web.
  • Publish content and deploy it on websites and implement security.
  • Use various data structures wisely.
  • Program applications in environments using database management systems.
  • Practice the techniques required to participate in all stages of systems development.
  • Communicate effectively with the various people who must intervene during the different stages of computer systems development.

Career opportunities

The world of information technology (IT) offers several opportunities for new graduates. Here are some jobs you could do:

Web developer

You will take care of the technical implementation and IT development of web applications. You will program the features that meet the client's needs for their web application and much more!

Database manager

You will become the reference for automating the organization, saving it and optimizing the use of data of the organization in which you will work.

Mobile developer

You will be able to create applications adapted to today's multiple devices and needs. It can be the mobile version of a website or a real application for smartphone and/or tablet.
